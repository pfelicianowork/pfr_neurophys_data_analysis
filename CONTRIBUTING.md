# Contributing to PFR Neurophys Data Analysis

Thank you for your interest in contributing! Please follow these guidelines:

## How to Contribute
- Fork the repository and create a new branch for your feature or bugfix.
- Write clear, well-documented code with comments and docstrings.
- Add or update tests for new features or bugfixes.
- Update documentation as needed.
- Submit a pull request describing your changes.

## Code Style
- Follow PEP8 for Python code.
- Use descriptive variable and function names.
- Document all public functions and classes.

## Issues
- Report bugs or request features via GitHub Issues.
- Include steps to reproduce and relevant logs or screenshots.
